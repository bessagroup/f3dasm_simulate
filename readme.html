Welcome to the documentation page of the simulation extension of the ‘Framework for Data-Driven Design and Analysis of Structures and Materials’ (f3dasm_simulate). Here you will find all information on installing, using and contributing to the Python package.

Warning

This package is not standalone and requires the :mod:`f3dasm` package to be installed. If you want more information on the f3dasm main package, go the the documentation page There you will find a getting started, statement of need, installation instructions and much more.
